# sidebar-for-dev

A terminal UI (TUI) sidebar for developers showing real-time: LiteLLM proxy cost, open GitHub PRs, and recent AI coding sessions (Claude Code, pi, opencode).

## Features

- **Cost panel** — 30-day stacked bar chart of LiteLLM proxy spend, broken down per model with configurable colors
- **PR panel** — All open PRs authored by you, grouped by repo, sorted by last update, with approval and check status badges
- **Session panel** — Recent Claude Code / pi / opencode sessions (past 3 days), token counts, cost, and AI-generated summaries (cached, via Haiku)
- **Settings editor** — In-TUI settings (`=` key), persisted to `~/.sidebar_for_dev/settings.json`
- **Auto-refresh** — Independent configurable intervals per panel (default 5 min)

## Requirements

- Node.js 20+
- `gh` CLI authenticated (`gh auth login`)
- `LITELLM_API_BASE` and `LITELLM_API_KEY` env vars (or set in settings)
- opencode optional — reads `~/.local/share/opencode/opencode.db` if present

## Install & Run

```bash
npm install
npm run dev
```

For a split-terminal setup (e.g. tmux):

```bash
# In one pane
npm run dev
```

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `1` | Switch to Cost panel |
| `2` | Switch to PRs panel |
| `3` | Switch to Sessions panel |
| `R` | Refresh current panel |
| `S` | Re-summarize selected session (Sessions panel) |
| `=` | Open settings editor |
| `↑↓` | Navigate list items |
| `q` | Quit |

## Settings (`~/.sidebar_for_dev/settings.json`)

```jsonc
{
  "refreshIntervals": { "cost": 300, "prs": 300, "sessions": 300 },
  "costChartDays": 30,
  "summaryModel": "claude-haiku-4-5",
  "litellm": { "apiBase": "", "apiKey": "" },
  "modelPricing": {
    "claude-sonnet-4-6": { "input": 3, "output": 15 },
    "claude-opus-4-8": { "input": 15, "output": 75 },
    "claude-haiku-4-5": { "input": 0.8, "output": 4 }
  },
  "modelColors": {
    "claude-sonnet-4-6": "cyan",
    "claude-opus-4-8": "magenta"
  },
  "ghProxy": "http://127.0.0.1:9000"
}
```

- `litellm.apiBase/apiKey`: auto-read from env if blank
- `modelPricing`: per 1M tokens; used to calculate session cost when not recorded by the agent
- `modelColors`: any undefined model gets a deterministic high-contrast color
- `ghProxy`: used for `gh` CLI calls via HTTP proxy (Bullish IP allowlist)

## Session summaries

Summaries are generated by `claude-haiku-4-5` via LiteLLM proxy and cached in `~/.sidebar_for_dev/summaries.json`. They are only re-generated when session content changes or when manually triggered with `S`.

## Architecture

```
src/
  app.tsx                     # Ink root, tabs, key handler, refresh loops
  config.ts                   # Settings load/save
  pricing.ts                  # Cost calculation, model color resolution
  data/
    litellm.ts                # LiteLLM /spend/logs API
    prs.ts                    # gh CLI wrapper + parser
    sessions/
      claude.ts  pi.ts  opencode.ts   # Per-tool session readers
      index.ts                # Merge + filter + sort
    summarize.ts              # Haiku summarizer + disk cache
  panels/
    CostPanel.tsx             # Stacked bar chart
    PrPanel.tsx               # PR list with badges
    SessionPanel.tsx          # Session list with summaries
    SettingsPanel.tsx         # In-TUI settings editor
```
